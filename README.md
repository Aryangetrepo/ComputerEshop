# COPA Smart Store - HTML CSS JavaScript E-Commerce Project

This is a professional mini e-commerce website for ITI COPA students. It includes:

- Responsive home page
- Product listing
- Search and category filter
- Add to cart, update quantity, remove item
- Checkout form validation
- LocalStorage based cart
- Commented source code

## How to Run
1. Extract the ZIP file.
2. Open `COPA-Ecommerce-Project/index.html` in Chrome/Edge.
3. Navigate to Products, add items to cart and test checkout.

No server installation is required.
